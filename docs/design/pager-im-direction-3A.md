# Pager IM 方向 3A（成员）— 抹茶视觉改版

来源：claude.ai/design 项目「Pager UI设计方向选择」的 `Pager IM 方向.dc.html`。
选定：**3A 成员**（AI 与人平级，同一条时间线，工具折成紧凑小卡片，授权是一张卡片）+ **先视觉改版**（重塑现有单人 app 外观，不改后端/多人）。

## 抹茶配色 token
- 背景：聊天体 `#eceadd`；头/输入栏 `#f5f4ea`
- 品牌绿：主/发送/用户气泡 `#86a15c`；深绿文字 `#62823c`；图标/链接 `#6a8544`；运行态 `#7f9a54`
- 琥珀：AI 名/授权 `#bd7f36`、`#e0a05a`；授权卡底 `#f6ead6` 边 `#ecd6ab`；mono 命令 `#8a6a2a`
- 深：文字/AI 头像底 `#333a2d`
- 奶白气泡：底 `#fffef9`，边 `#e9e7d7` / `#efe6cf`
- 文字：主 `#333a2d`，次 `#5f6653`，弱/占位 `#a9ad97`
- 工具小卡片：底 `#f3f1e4`，边 `#e5e2d0`，mono 文字 `#4d5c34`，meta `#a9ad97`

## 3A 布局
- 用户气泡：绿底白字，右对齐，右上角 5px（其余 16px）
- AI/他人气泡：奶白，左对齐，左上角 5px；上方一行发送者名（AI 名用琥珀 `#bd7f36` 粗体「百姓AI」）
- AI 头像：`#333a2d` 圆 + `#e0a05a` 1.5px 边，内嵌橙色原子/轨道图标
- 工具调用 → 紧凑 chip：图标 + mono「Tool · 标题」+ 行数/meta，点开看 detail
- 权限请求 → 琥珀卡：盾图标 + `#bd7f36`「权限请求」+ mono 命令 + 允许(绿底白)/拒绝(奶白底 `#e5d3ad` 边 `#8a7a4a` 字)
- 状态：日期分隔 `#e2e0d0` 底 `#b0b39d` 字胶囊；运行态 `#eef1e0` 底 `#62823c` 字 + 脉冲圆点
- 输入栏：奶白胶囊 radius 22，占位 `#a9ad97`，圆形绿发送按钮
